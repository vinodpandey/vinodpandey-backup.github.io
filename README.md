#Documentation

####Installing required dependencies
```sh
gem install jekyll bundler
```

####checkout code
```sh
git clone git@github.com:vinodpandey/vinodpandey.github.io.git
cd vinodpandey.github.io
bundle install
bundle exec jekyll serve
```
