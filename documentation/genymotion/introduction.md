# Genymotion

## Installing Google Play Service
```
https://guides.codepath.com/android/Genymotion-2.0-Emulators-with-Google-Play-support

Files Downloaded:
Genymotion-ARM-Translation_v1.1.zip
gapps-lp-20141109-signed.zip

Tested With:
- Genymotion 2.8
- Nexus 4 - 5.0.0 - API 21
- Install both ARM Translation Installer & Android 5.0 Google Play Services APK

Google+ and Google Play Service will show popup continuosly
Keep calm and continue login with Google ID
Select update for both Google+ and Google Play Service

```


## Copying data from Genymotion to Host machine
```
Using Android Device Monitor (part of android studio)
https://developer.android.com/studio/profile/monitor.html

monitor
File Explorer > /data/data
Select directory - On Top Right Toolbar > Pull file from device

```
